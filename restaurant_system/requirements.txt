# 餐饮点单系统依赖包
# Python 3.8+ 内置模块，无需额外安装第三方包

# 核心依赖（Python内置）:
# - tkinter (GUI框架)
# - dataclasses (数据类)
# - typing (类型提示)
# - decimal (精确货币计算)
# - datetime (日期时间)
# - uuid (唯一ID生成)
# - enum (枚举类型)
# - abc (抽象基类)

# 如需数据库支持，可选安装:
# sqlite3 (内置)
# psycopg2-binary==2.9.9  # PostgreSQL
# pymysql==1.1.0  # MySQL
